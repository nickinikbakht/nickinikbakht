## Selected Research Projects

### CombOpt-Zero (Master’s Thesis)
Developed a hybrid framework combining graph neural networks and Monte Carlo Tree Search for solving NP-hard combinatorial optimization problems.  
Evaluated on benchmark instances including Maximum Independent Set and Vertex Cover, with comparisons against exact solvers.

### Multi-Objective Optimization Framework (BRADO Extension)
Extended a metaheuristic optimization framework for multi-objective settings, enabling Pareto-optimal solution generation under conflicting engineering constraints.

### Industrial Energy Scheduling Optimization
Formulated and implemented constrained optimization models for real-time industrial energy allocation under demand-side constraints.

### Predictive Maintenance for Industrial Systems
Developed machine learning models for fault detection and Remaining Useful Life estimation using multivariate industrial sensor data.
